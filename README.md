# Olist Customer Analytics and Churn Prediction

A SQL and machine learning project using the Olist Brazilian E-Commerce dataset.

This project explores e-commerce customer behaviour using SQLite, SQL queries, pandas, and machine learning. The goal is to practise working with relational databases, extracting useful features with SQL, and later using those features to build a customer churn prediction model.

## Project Overview

The project currently focuses on building a SQLite database from the Olist CSV files and running SQL queries from Python.

## Dataset

The dataset used in this project is the Olist Brazilian E-Commerce dataset from Kaggle.

Source: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

The dataset contains multiple related tables, including customers, orders, products, sellers, payments, reviews, and order items.

## Project Purpose

This project was created to practise the full data workflow used in many real-world machine learning projects:

```text
Raw data
↓
Database
↓
SQL queries
↓
Feature engineering
↓
Python analysis
↓
Machine learning
```
